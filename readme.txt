A theme for Wordpress.

== Description ==

A simple HTML 5 theme for Wordpress, built with [Twitter Bootstrap](http://twitter.github.com/bootstrap/index.html), [Font Awesome](http://fortawesome.github.com/Font-Awesome/), and [Open Sans](http://opensans.com/).

Features optional right-hand sidebar, custom header, custom menu, and widgets in the footer.

== Installation ==

1. Upload the 'graphite' folder to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Appearance -> Themes' menu in WordPress.
3. Configure the theme through the 'Customize' link on the Themes page in WordPress.
